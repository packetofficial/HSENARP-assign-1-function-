PROGRAMMING ASSIGNMENT-1
(BASED ON THE CHAPTER USER DEFINED FUNCTIONS)


Q1. Write a program to input five numbers from the user and display the greatest number using a function that returns the greater of two numbers.


Q2. Write a program to display all the prime palindrome numbers within the range m to n where m is the lower limit and n is the upper limit. If there is no prime palindrome number within the given range then display an appropriate message


Q3. Write a program using two or more functions to display all the special numbers less than a given limit. A number is said to be a special number if the sum of the factorials of individual digits is equal to the number itself. e.g. : 145= 1!+ 4! +5! = 1+ 24+ 120 = 145


Q4 Write a program to display all the pairs of twin primes less than a given limit. Twin primes means two prime numbers with a difference of two. For example if the limit is 12 then the output should be Pairs of twin primes less than 12 (3,5) (5,7)


Q5. Write a program to display all the Niven numbers less than a given limit. If a number is a multiple of its sum of digits then it is called a Niven number. For example, 81 is Niven number.

Q9. Define a class Num with three overloaded functions named compute and a main function i) first overloaded function should return the sum of digits of a number ii) second overloaded function should return the product of digits of a number iii) third overloaded function should return the reverse of a number iv) void main() in which input a number say n and add the sum of digits and the product of digits of n . If the result is equal to n then display a message “special number” else find the reverse of n and display it. For e.g. n=59 sum of digits = 14 product of digits = 45 sum of digits + product of digits = 14+45 = 59 So 59 is a special number
